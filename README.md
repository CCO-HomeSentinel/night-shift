## Installation

```bash
  pip install -r requirements.txt
```



## Environment Variables

To run this project, you will need to add the following environment variables to your .env file.

`MYSQL_HOST`

`MYSQL_PORT`

`MYSQL_DATABASE`

`MYSQL_USERNAME`

`MYSQL_PASSWORD`

`BUCKET_NAME`

`FLASK_PORT`

`FLASK_ENDPOINT`

`FLASK_DEBUG`

`ENABLE_LOGS`

`INTERVALO_BACKUP_LOGGER`

`JWT_SECRET_KEY`

`JWT_ALGORITHM`

`SECRET_KEY`

`JWT_TIME`
